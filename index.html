概要

  • JavaScript から呼べる、 Hot Soup Processor 風の命令セットライブラリです
  • Canvas を利用し、グラフィック描画命令が実行可能
  • UIオブジェクト (button, chkbox, input) 対応
  • 作りかけ

ポリシー

  • HSP「風」のインターフェースを提供するもので、完全な互換ではありません
  • グローバル変数が散らかりますが、手軽な実行を優先しています
  • 使用者が、ある程度 JavaScript を書けることを想定しています

制限事項

  • モジュール非対応
  • wait / await 非対応
  • その他色々...
